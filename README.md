# Lemnova CRM Jurídico

## Sobre o projeto

O **Lemnova** é uma plataforma de gestão jurídica desenvolvida para organizar rotinas de escritórios de advocacia, centralizando informações de clientes, processos, documentos, tarefas e atividades administrativas em um único ambiente.

O projeto passou por uma etapa de evolução de identidade e estrutura, deixando de utilizar a marca **Jures One** para assumir o novo conceito **Lemnova**, representando uma nova fase de desenvolvimento, organização e expansão da plataforma.

---

# Atualização de Marca — Jures One → Lemnova

A atualização do projeto envolveu uma reformulação completa da identidade visual e da apresentação da aplicação.

Principais mudanças realizadas:

* Alteração da identidade da aplicação para **Lemnova CRM Jurídico**
* Atualização do nome do projeto nos ambientes de desenvolvimento e publicação
* Atualização do deploy da aplicação para a nova identidade
* Ajustes de apresentação visual e elementos de marca
* Organização da estrutura para futuras evoluções do sistema

---

# Objetivo da atualização

A mudança para Lemnova representa uma nova etapa do projeto, com foco em:

* Criar uma identidade própria para a plataforma
* Melhorar a percepção profissional do produto
* Preparar a aplicação para futuras integrações e expansão
* Construir uma base mais próxima de uma solução SaaS jurídica
